# Hotel Management System

## 📌 Project Overview

The **Hotel Management System** is a Java-based console application designed to simulate basic hotel operations such as customer management, room management, and reservation booking.
The project demonstrates object-oriented programming principles and a layered architecture to keep the code organized and maintainable.

This system allows users to manage hotel data through a simple command-line interface, making it useful for learning Java application structure and backend logic.

## 🚀 Features

* Create and manage customer records
* View available rooms
* Book room reservations
* Manage hotel data efficiently
* Console-based user interface
* Modular architecture with separate layers

## 🏗️ Project Structure

The project follows a layered structure:

Hotel-Management-System
│

├── api        # Handles interaction between UI and services

├── service    # Business logic of the application

├── model      # Data models such as Customer, Room, Reservation

├── ui         # Console user interface

└── main       # Entry point of the application

## 🛠️ Technologies Used

* **Java**
* **Object-Oriented Programming (OOP)**
* **Console-based Interface**
